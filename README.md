# churn_analysis_
📌Project Title
Customer Churn Prediction

📌 Project Overview
The Customer Churn Analysis Dashboard analyzes customer behavior to identify patterns and key factors contributing to customer churn. The dashboard provides interactive visual insights that help businesses understand why customers leave and how retention strategies can be improved. By analyzing customer demographics, service usage, and subscription details, the project enables data-driven decision making.

🎯 Purpose
Customer churn is a major challenge for many businesses. This project aims to:
- Analyze customer data to identify churn patterns
- Highlight factors affecting customer retention
- Provide insights through an interactive dashboard
- Support better business decisions to reduce churn

🛠️ Tech Stack
Power BI – Dashboard and visualization
Microsoft Excel – Data cleaning and preprocessing
DAX – Data calculations and metrics
Power Query – Data transformation
SQL – Data querying and analysis
Random Forest – Churn prediction model

📂 Data Source
The dataset used in this project is a Customer Churn dataset containing information such as:
- Customer demographics
- Churn by age
- Churn by enure group
- Churn by gender
- Churn by contact
- Payment methods
- Churn status
This dataset represents telecom-style customer behavior commonly used for churn analysis.

📈 Dashboard Features
Churn Rate Analysis – Overall churn percentage
Customer Segmentation – Insights by gender, tenure, and contract type
Payment Method Analysis – Impact of payment methods on churn
Service Usage Insights – Relationship between services and churn rate

⭐ Highlights
End-to-End Data Analytics Pipeline -
The project covers the complete workflow from data extraction using SQL, data preprocessing, machine learning model building, and visualization through Power BI dashboards, demonstrating a full data analytics lifecycle.

Machine Learning-Based Churn Prediction -
A Random Forest classification model was implemented to predict whether a customer is likely to churn based on behavioral and demographic features, helping businesses identify at-risk customers.

Identification of Key Churn Drivers -
Feature importance analysis was used to determine the most influential factors contributing to churn, such as contract type, tenure, payment method, and service usage patterns.

Interactive Business Intelligence Dashboard -
A Power BI dashboard was created to present insights visually, allowing stakeholders to explore churn trends, customer segments, and key metrics through interactive filters and charts.

Actionable Business Insights -
The analysis provides practical insights that can help organizations improve customer retention strategies, reduce churn rate, and enhance customer satisfaction.
